# ddd-event-sourcing-snapshot
Domain Driven Design（DDD）实现事件溯源（Event Sourcing）的快照版，通过snapshot加速聚合根重建，基于ddd-archetype。

本项目是《悟道领域驱动》一书的随书案例。
